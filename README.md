# Productivity Hub – Multi-Step Task Manager

A modern and responsive web application developed using HTML, CSS, and JavaScript as part of the ApexPlanet Web Development Internship.

## 📌 Project Overview

Productivity Hub is a multi-step task management application that guides users through a simple workflow:

1. Fill out a Contact Form
2. Manage and organize tasks dynamically
3. View a Success Page after task completion

The application demonstrates form validation, DOM manipulation, responsive design, and interactive user experiences using core front-end technologies.

## ✨ Features

- Multi-Step User Interface
- Contact Form with Validation
- Dynamic Task Addition and Deletion
- Interactive Success Page
- Responsive Design for Mobile, Tablet, and Desktop
- Clean and Modern User Interface
- Smooth Navigation Between Steps
- Real-Time User Interaction with JavaScript

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
